## Controls

### Character Movement:

A-Move Backgwards.
S-Crouch.
D-Move Forward.
W-Jump.

Character is able to double jump if W is pressed after jumping once.

### Attack

Mouse right click- Shoot.


## Debug

F1-Change level(to level 1).
F2-Change level(to level 2).
F3-Reset current level.
F5- Save Game.
F6-Load Game.
F10- GodMode.
F11-Change frame cap.
